# expo_music_app
expo_music_app


For app multi platfrom test : npx expo start --tunnel

1. Clone the repository
git clone git@github.com:sohchy/music-reactnative-prototype.git

2. Enter directory
cd expo_music_app

3. Install dependencies (using pnpm)
pnpm install

4. Start the development server
To start the dev server (QR code)
npx expo start

To run directly on Android
npx expo run:android

To run directly on iOS
npx expo run:ios

To clear cache (if you face issues)
npx expo start -c
